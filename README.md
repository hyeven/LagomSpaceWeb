📌 프로젝트 소개
- LagomSpaceWeb은 간단한 커뮤니티 웹 애플리케이션입니다.
- 웹 취약점 진단을 목적으로 개발한 서비스 입니다.

<br>
  
🌟 주요 기능
- 사용자 회원가입 및 로그인
- 게시글 작성, 수정, 삭제
- 프로필 페이지
- 관리자 페이지
- 반응형 UI (모바일 지원)

<br>

🛠️ 기술 스택
- 프론트엔드: HTML, CSS, JavaScript
- 백엔드: Python (Flask)
- 데이터베이스: SQLite
- 버전 관리: Git & GitHub
